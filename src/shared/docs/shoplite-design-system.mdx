# ShopLite Design System

ShopLite is a frontend-only portfolio commerce project with a dark premium visual direction.

## Visual direction

- Aetheric commerce
- Digital luxury
- Dark premium UI
- Glass panels
- Electric violet
- Neon cyan
- Accent pink
- Glow shadows

## Current component foundation

- Button
- ButtonLink
- Badge
- Card
- Input
- SkeletonBlock
- PageLoadingState

## Storybook goal

Storybook is used as a visual documentation layer for the reusable UI system. It should help validate the component variants before they are used in catalog, cart, checkout and account flows.

## Frontend-only constraint

ShopLite does not use real payments, backend secrets or private API keys. Any checkout, auth or order flow is intentionally simulated for portfolio purposes.

## Microinteractions

ShopLite uses subtle motion to make the commerce flow feel more premium:

- `interactive-lift` for clickable cards
- `interactive-press` for buttons
- `soft-border-glow` for premium product surfaces
- `skeleton-shimmer` for loading states
- reduced motion support through `prefers-reduced-motion`

Motion should be subtle and should not block usability.
